# MotrixLab

MotrixLab 是一个为机器人训练设计的机器学习通用架构。它基于 MotrixSim 仿真平台搭建，可以实现在 CPU 或 GPU 上的策略训练，为各类操作系统和硬件设备使用者，提供灵活易用的训练能力。

## 主要特性

-   **跨软件平台能力**: 支持 Windows 和 Linux 操作系统环境
-   **支持 CPU 仿真**: 使用 MotrixSim 在 CPU 上进行物理仿真，对 GPU 的要求大幅降低

## 适用场景

-   机器人控制算法开发和测试
-   强化学习环境构建
-   教育和研究
